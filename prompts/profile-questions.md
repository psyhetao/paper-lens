# 个人研究档案设置 (paper-lens)

您好！为了提供最精准的文献分析，我需要了解您的研究背景。请回答以下问题：

1. **学科领域**：您从事的研究领域是什么？（如：AI、医学、社会学等）
2. **研究方向**：您目前关注的具体课题或方向是什么？
3. **核心关注点**：您阅读文献时最看重什么？（如：算法细节、实验结论、理论推导等）
4. **常用关键词**：哪些术语能代表您的研究兴趣？
5. **语言偏好**：您希望笔记使用哪种语言？（中文/英文/跟随原文）
6. **笔记软件**：您是否使用特定的笔记工具？（如：Obsidian, Notion, Logseq）
7. **APA 格式**：是否需要自动生成 APA 引用？

*您可以随时输入「更新研究档案」来修改这些信息。*
